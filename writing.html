Typography

Effective UX design begins with the simplicity of type.

Example

This is an H1 header.
THIS IS AN H3 TITLE. IT'S IN ALL CAPS.
This is a label. It's not in all caps.
This is a value
This is a link.

Markup

<h1>This is an H1 title.</h1>
<h3>This is an H3 title. It is in all caps.</h3>
<h3>This is a label. It is not in all caps.</h3>
<h1>This is a value.</h1>
<a>This is a a link</a>

Style

h1,h3,p,a {
font-family: Helvetica, Arial, sans-serif;
}

h1 {
font-size: 16px;
color: #006FAE;
font-weight: bold;
}

h3 {
font-size: 12px;
color: #666;
font-weight: bold;
}

p {
font-size: 12px;
color: #666;
font-weight: bold;
}

p {
font-size: 12px;
color: #666;
}

Commentary

Remember to use typography consistently. Consistent typography results in effective communication of instruction, data, and feedback to the user. There must be proper hierarchy to what is shown on the screen. Remember, not everything needs to be bold. Intuition is created by structure, not by making everything stand out on the screen.
